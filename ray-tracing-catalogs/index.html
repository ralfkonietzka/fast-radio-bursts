Fast Radio Burst Ray-tracing Catalogs

This page serves as a centralized platform for accessing the ray-tracing maps and catalogs
generated by Konietzka et al. in 2025. Below, you will find links to the main data samples and additional ones.
Each catalog consists of a dispersion measure (DM) dataset of dimension Nrays x Nredshifts entitled DMvalues.
Nrays is the number of rays provided. Nredshifts is the number of redshift bins.
The redshift values corresponding to the redshift bins can be found in the table entitled redshifts.
If you use these data, we ask you to cite Konietzka et al., 2025.

NOTE: The catalogs below are scaled-down versions of the final datasets.
The full-size catalogs will be available on this page soon.
